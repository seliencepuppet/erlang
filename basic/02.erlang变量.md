
## 变量

怎样才能把一个命令的结果保存起来, 以供后面使用呢? 这就是变量的职责所在. 下面是一个例子:

```erl
1> X = 123456789.
123456789
```

这是什么意思呢? 首先, 我们向变量X赋了一个值, 然后shell打印出了变量的值.

#### 说明   所有的变量都必须以大写字母开头.

如果要查看一个变量的值, 那么只需要输入变量的名字即可:

```erl
2> X.
123456789
```

现在X有了值, 你可以这样使用它:

```erl
3> X*X*X*X.
232305722798259244150093798251441
```
